THE SCIENCE BEHIND MIXBOT

หลักการทางวิทยาศาสตร์และคณิตศาสตร์

Inverse Kinematics

การคำนวณย้อนกลับจากพิกัด (X, Y, Z) เพื่อหาองศาการหมุนของ Servo ทั้ง 6 ตัว โดยใช้ตรีโกณมิติระดับสูง เพื่อให้ตำแหน่งการเคลื่อนที่แม่นยำในระดับ 1.0 มม.

$\theta = \arctan2(y, x)$

PID Control Algorithm

ระบบควบคุมแบบป้อนกลับ (Feedback Control) ที่ใช้ในระบบชั่งน้ำหนัก Load Cell เพื่อป้องกันการจ่ายของเหลวเกินขนาด (Overshoot) และหยุดจ่ายได้ทันทีที่น้ำหนักถึงเกณฑ์

Fluid Dynamics Analysis

วิเคราะห์อัตราการไหลของของเหลวตามแรงโน้มถ่วง เพื่อคำนวณเวลาในการเปิด-ปิดวาล์วที่สัมพันธ์กับความหนืดของน้ำเชื่อมชนิดต่างๆ

แขนกลอัจฉริยะ (Smart Robot Arm)

หัวใจหลักในการเคลื่อนที่และหยิบจับภาชนะ ออกแบบมาเพื่อความคล่องตัวและแม่นยำสูงในพื้นที่จำกัด

เครื่องเขย่าความเร็วสูง (Shaking Machine)

ระบบเขย่าเชิงกลที่ออกแบบมาเพื่อสกัดรสชาติและผสมส่วนผสมให้เข้ากันอย่างสมบูรณ์แบบในเวลาที่กำหนด

SENSOR TECHNOLOGY

หลักการทำงานของเซนเซอร์อัจฉริยะ

Infrared Proximity Sensing

หลักการ: ใช้การแผ่รังสีอินฟราเรดจากตัวส่ง (Emitter) และตรวจวัดความเข้มของรังสีที่สะท้อนกลับมายังตัวรับ (Receiver) เพื่อยืนยันตำแหน่งเชิงพื้นที่ของวัตถุ (Object Detection) บนแท่นปฏิบัติการก่อนเริ่มกระบวนการถัดไป

Ultrasonic Distance Measurement

หลักการ: อาศัยการคำนวณระยะเวลาการเดินทางของคลื่นเสียงความถี่สูง (Time of Flight) ที่สะท้อนจากผิววัตถุกลับมายังเซนเซอร์ เพื่อตรวจวัดระดับความพูนของน้ำแข็งในกระบอกแบบ Non-contact โดยมีความแม่นยำสูงในระดับมิลลิเมตร

Load Cell Piezoresistive Sensing

หลักการ: ตรวจวัดการเปลี่ยนแปลงความต้านทานไฟฟ้าของ Strain Gauge เมื่อได้รับแรงกดจากการเติมของเหลว (Weight-based Feedback Control) ทำงานร่วมกับฟังก์ชัน Tare เพื่อขจัดค่าคงที่ของภาชนะ ทำให้ได้ปริมาตรของส่วนผสมที่เที่ยงตรงตามหลักจลนศาสตร์

PROJECT JOURNEY

เส้นทางการพัฒนาโครงงาน

ระยะที่ 1: การกำหนดปัญหาและการทบทวนวรรณกรรม

ศึกษาเอกสารและงานวิจัยที่เกี่ยวข้อง (Literature Review) เพื่อวิเคราะห์ความเป็นไปได้ทางเทคนิค กำหนดขอบเขต และวางสถาปัตยกรรมระบบหุ่นยนต์อัจฉริยะ

ระยะที่ 2: การออกแบบเชิงวิศวกรรมและการคำนวณจลนศาสตร์

พัฒนาแบบจำลอง 3 มิติเชิงทัศนวิสัย (3D Modeling) และคำนวณทางจลนศาสตร์ (Kinematics) เพื่อคัดเลือกอุปกรณ์และระบบอิเล็กทรอนิกส์ให้สอดคล้องกับมาตรฐานทางวิศวกรรม

ระยะที่ 3: การบูรณาการระบบและการประมวลผลคำสั่ง

ดำเนินการติดตั้งโครงสร้างกลไกเชิงกล พัฒนาระบบควบคุมไมโครคอนโทรลเลอร์แบบ Real-time และส่วนต่อประสานกับผู้ใช้ (UI) ให้ทำงานประสานกันอย่างสมบูรณ์

ระยะที่ 4: การสอบเทียบ การวิเคราะห์ผล และการประเมินประสิทธิภาพ

ดำเนินการสอบเทียบเครื่องมือ (Calibration) เพื่อหาค่าความคลาดเคลื่อน วิเคราะห์เสถียรภาพในการริน และประเมินมาตรฐานความปลอดภัยตามหลักการยศาสตร์
